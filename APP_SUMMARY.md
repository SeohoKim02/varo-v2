# Varo V2 제출 요약

## 한 줄 설명

Varo V2는 악성재고를 찾아 점포 간 이동 경로를 추천하고, 추천 근거와 비교 검증을 한 화면 흐름으로 보여주는 Streamlit 데모다.

## 해결하려는 문제

점포별 과잉·부족 재고를 따로 확인하던 과정을 연결해, 폐기 위험이 있는 재고를 어디로 얼마나 이동할지 빠르게 검토할 수 있게 한다.

## 핵심 기능 5개

1. 엑셀 업로드와 기본·DQN 샘플 로드
2. 악성재고 판단과 점포 간 이동 후보 생성
3. VHS 중심 추천 및 Greedy·DQN·Pareto 비교
4. Top 경로 네트워크 시뮬레이션과 선택 경로 상세
5. DQN 원본·균형형 학습 비교와 DIRECT/VIA_DC 경로 상세

## 알고리즘 구조

VHS가 최종 추천 우선순위를 정한다. Greedy는 단순 기준의 비교선, Pareto는 후보 간 보조 검증으로 사용한다. DQN은 자동 실행하지 않으며 정상 상태이고 현재 데이터 signature와 일치할 때만 최대 8%의 참고값으로 사용한다.

## DQN 원본 vs 균형형 비교 결과 요약

원본 10개 샘플에서는 target action 라벨 편향을 진단했다. 원본 수치 특성은 유지하고 파생 라벨만 재분배한 균형형 10개를 별도로 학습해 데이터 품질과 학습 안정성을 비교했다. 검토가 필요한 원본 결과는 최종 추천에 넣지 않고, 조건을 통과한 결과만 제한적으로 참고한다.

## 교수님 피드백 반영 요약

밝은 테마와 결과 중심 UI로 설명을 줄였고, 네트워크 배치와 차량·경로 가독성을 개선했다. VHS 자동 가중치, 민감도·신뢰도, Greedy·DQN·Pareto 비교와 제한 탐색 기반 검증을 연결해 추천 근거를 확인할 수 있게 했다.

## 현재 완성 상태

알고리즘 연결, DQN 원본·균형형 10개 비교, 10점포·2DC 샘플, 자체 경로 단계, 배포 의존성 분리와 자동 테스트가 준비되어 있다.

## 남은 확인 사항

- 실제 제출 브라우저에서 5개 페이지의 배치와 Console error를 육안 확인
- 경로 상세에서 DIRECT·VIA_DC 이동 단계와 DC01·DC02 구분 확인
